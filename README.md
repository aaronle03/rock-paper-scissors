# rock-paper-scissors
First Usage of JavaScript, creating a Rock-Paper-Scissors Game

HOW-TO-PLAY
1. Load Website Up
2. Click your choice
3. The game plays up to 5 rounds
4. Click the botton RESET GAME button to play again!
